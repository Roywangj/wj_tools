### 旧 SSH 算法主机说明

`10.106.15.118` 与 `10.106.15.96` 仅接受旧版 SSH 主机密钥/公钥算法。客户端连接时需在 `~/.ssh/config` 中为这两台机器追加以下选项（或在命令行用 `-o` 传入）：

```sshconfig
HostKeyAlgorithms +ssh-rsa
PubkeyAcceptedKeyTypes +ssh-rsa
```
